# Image Gallery & Slideshow

This project is a simple Image Gallery and Automatic Slideshow built using HTML and CSS.

## Features

- Clickable thumbnail images
- Enlarged image preview
- Automatic image slideshow
- Smooth CSS animations
- Responsive design

## Technologies Used

- HTML5
- CSS3
